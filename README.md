# Sarthak Pawar — Portfolio

Personal portfolio built with React, TypeScript, Vite, and Tailwind CSS.

## Stack

- **React 19** + **TypeScript**
- **Vite** — build tool
- **Tailwind CSS v4**
- **Framer Motion** — animations
- **Spline** — 3D scene
- **Three.js** — shader effects

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Deployed on [Vercel](https://vercel.com). Push to `main` to trigger a deployment.
